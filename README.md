# Phang Lehenn Portfolio Website

Responsive portfolio website built with **React + Vite + TypeScript**.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Free hosting options

- GitHub Pages
- Vercel
- Netlify

## Deploy with Vercel

1. Push this project to GitHub.
2. Open Vercel and import the GitHub repository.
3. Use `npm run build` as the build command.
4. Use `dist` as the output directory.

Privacy note: the public site uses email, phone, GitHub, and general location only. The full street address from the resume was intentionally left out.
